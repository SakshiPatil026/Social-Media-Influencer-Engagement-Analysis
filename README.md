# Social-Media-Influencer-Engagement-Analysis
This project analyzes social media influencer performance using engagement metrics.
The goal is to identify top performing influencers based on audience interaction rather than follower count.
The dataset contains influencer post data including likes, comments, shares, and followers.
Using Python (Pandas & Matplotlib), the data is cleaned, processed, and visualized to generate insights.

Objectives:
Clean messy social media dataset
Calculate engagement rate
Rank influencers based on performance
Visualize top influencers
Generate business insights

Tools & Technologies:
Python
Pandas
Matplotlib
CSV Dataset

Steps Performed:
Loaded dataset using Pandas
Cleaned inconsistent date and numeric values
Removed invalid records (zero followers)
Calculated engagement rate
Identified top 10 influencers
Created bar chart visualization
